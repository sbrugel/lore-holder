Lore Keeper

An online solution to maintaining your world-building information, including universes, characters, locations, and lore.

Getting started is simple - just sign in above with your Google account!

Version 1.0.0 (January 2025)

Welcome to the first version of the site! All of the core functionality works now - you can create/edit/delete worlds, characters, places, and stories, and set numerous properties for each of these. Certain things are implemented visually, but don't function fully yet (i.e. gallery links for characters), but these will be added in soon.

Current features as of this release:

  • Sign up and login using Google SSO, and stay logged in between sessions
  • Create new worlds, providing a name, brief description, extended description, and optional image
  • Within worlds, create characters and places with extensive details, and even create your own details to display on their pages
  • Within worlds, create stories that have customizable text and image modules. You can even set an "order" for your stories by giving each story a previous and next story. You can also give character tags to a story to better organize them, i.e. if you have divergent/parallel storylines.
  • Freely edit and delete all of these objects that you own

In the next major release, I aim to do the following:

  • Some final code cleanup - everything works but I'd like to keep things tidy
  • Ability to upload images instead of having to paste links (probably using the imgur API)
  • World privacy settings. Currently, only the world owner can view the world. You will be able to make worlds public, so others can view but not edit world contents.
  • Ability to manually reorder characters, places, stories, and story modules. Currently, these are ordered either by name or creation date, whichever one worked best for each object type.
  • Shared custom details - ensure that every character and place has a "base set" of details that you can work with
  • Some more stuff down the road when I get time :)

In the meantime, feel free to report any issues or suggestions on the repository's issues tracker.
